# CreativeSpy — плагин для Minecraft (Paper/Spigot 1.21.1)

## Что делает плагин
- Записывает, когда игрок **заходит** и **выходит** из креативного режима.
- Записывает, какие предметы игрок **брал из креативного инвентаря** (материал и количество).
- Данные хранятся в файлах и переживают перезапуск сервера.
- Команда `/c spec <ник>` показывает последние 50 записей по игроку.
- Право на использование команды: `creativespy.spec` (по умолчанию есть у оперов).

---

## Как получить готовый .jar — просто, без установки программ на компьютер

Сборка будет происходить бесплатно на серверах GitHub, тебе нужно только загрузить файлы через браузер. Кода писать не придётся.

### Шаг 1. Заведи аккаунт на GitHub
Если аккаунта нет — зайди на https://github.com/signup и зарегистрируйся (это бесплатно, займёт минуту).

### Шаг 2. Создай новый репозиторий
1. Зайди на https://github.com/new
2. В поле "Repository name" впиши, например, `CreativeSpy`
3. Ниже выбери **Private** (чтобы никто чужой не видел) — не обязательно, но так спокойнее
4. Нажми зелёную кнопку **Create repository**

### Шаг 3. Загрузи файлы плагина
1. Распакуй архив `CreativeSpy.zip`, который я тебе прислал, у себя на компьютере — внутри будет папка `CreativeSpy` со всеми файлами.
2. На странице твоего нового репозитория на GitHub нажми **"uploading an existing file"** (или "Add file" → "Upload files").
3. **Перетащи в окно браузера всю папку `CreativeSpy`** целиком (её содержимое — `pom.xml`, `README.md`, папки `src` и `.github`). GitHub сам сохранит структуру папок при перетаскивании.
4. Внизу нажми зелёную кнопку **Commit changes**.

### Шаг 4. Дождись сборки
1. Наверху страницы репозитория открой вкладку **Actions**.
2. Через несколько секунд там появится процесс сборки (жёлтый кружок → потом зелёная галочка, это занимает 1-2 минуты).
3. Когда появится зелёная галочка — кликни на этот запуск (называется "Build plugin").
4. Внизу страницы будет раздел **Artifacts**, там файл **CreativeSpy-jar** — нажми, чтобы скачать. Внутри скачанного zip-архива будет сам `CreativeSpy.jar`.

Если появился красный крестик вместо галочки — значит где-то ошибка при загрузке файлов (например, папки перепутались). Пришли мне скриншот вкладки Actions, помогу разобраться.

## Установка на сервер
1. Убедись, что сервер на **Paper** (или Spigot) версии **1.21.1**.
2. Положи скачанный `CreativeSpy.jar` в папку `plugins/` на сервере.
3. Перезапусти сервер.
4. В консоли сервера должно появиться сообщение `CreativeSpy включён!`.

## Использование
```
/c spec Steve
```
Покажет историю входов/выходов из креатива и взятых предметов для игрока Steve.

## На заметку
- Событие взятия предмета из креатива в Minecraft срабатывает довольно часто, поэтому в логах иногда могут попадаться немного лишние или повторяющиеся записи — это особенность самой игры, а не ошибка плагина.
- Если что-то пойдёт не так на любом из шагов — просто напиши мне, что случилось (можно приложить скриншот), разберёмся вместе.
